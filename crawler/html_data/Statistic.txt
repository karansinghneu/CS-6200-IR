It is calculated by applying a <a href="/wiki/Function_(mathematics)" title="Function (mathematics)">function</a> (statistical <a href="/wiki/Algorithm" title="Algorithm">algorithm</a>) to the values of the items of the <a href="/wiki/Sample_(statistics)" title="Sample (statistics)">sample</a>, which are known together as a <a href="/wiki/Data_set" title="Data set">set of data</a>.\n</p><p>More formally, <a href="/wiki/Statistical_theory" title="Statistical theory">statistical theory</a> defines a statistic as a function of a sample where the function itself is independent of the sample\'s distribution; that is, the function can be stated before realization of the data. The term statistic is used both for the function and for the value of the function on a given sample.\n</p><p>A statistic is distinct from a <a href="/wiki/Statistical_parameter" title="Statistical parameter">statistical parameter</a>, which is not computable in cases where the population is infinite, and therefore impossible to examine and measure all its items. However, a statistic, when used to estimate a <a href="/wiki/Statistical_parameter" title="Statistical parameter">population parameter</a>, is called an <a href="/wiki/Estimator" title="Estimator">estimator</a>. For instance, the <i><a class="mw-redirect" href="/wiki/Sample_mean" title="Sample mean">sample mean</a></i> is a statistic that estimates the <i>population mean</i>, which is a parameter.\n</p><p>When a statistic (a function) is being used for a specific purpose, it may be referred to by a name indicating its purpose: in <a href="/wiki/Descriptive_statistics" title="Descriptive statistics">descriptive statistics</a>, a <a class="mw-redirect" href="/wiki/Descriptive_statistic" title="Descriptive statistic">descriptive statistic</a> is used to describe the data; in <a href="/wiki/Estimation_theory" title="Estimation theory">estimation theory</a>, an estimator is used to estimate a parameter of the distribution (population); in <a href="/wiki/Statistical_hypothesis_testing" title="Statistical hypothesis testing">statistical hypothesis testing</a>, a <a href="/wiki/Test_statistic" title="Test statistic">test statistic</a> is used to test a hypothesis. However, a single statistic can be used for multiple purposes &#8211; for example the sample mean can be used to describe a data set, to estimate the population mean, or to test a hypothesis.\n</p>\n<div class="toc" id="toc"><input class="toctogglecheckbox" id="toctogglecheckbox" role="button" style="display:none" type="checkbox"/><div class="toctitle" dir="ltr" class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Statistic&amp;action=edit&amp;section=1" title="Edit section: Examples">edit</a><span class="mw-editsection-bracket">]</span></span></h2>\n<p>In calculating the <a href="/wiki/Arithmetic_mean" title="Arithmetic mean">arithmetic mean</a> of a sample, for example, the algorithm works by summing all the <a href="/wiki/Data" title="Data">data</a> values observed in the sample and then dividing this sum by the number of data items. This single measure, the mean of the sample, is called a statistic; its value is frequently used as an estimate of the mean value of all items comprising the population from which the sample is drawn. The <a class="mw-redirect" href="/wiki/Population_mean" title="Population mean">population mean</a> is also a single measure; however, it is not called a statistic, because it is not obtained from a sample; instead it is called a population parameter, because it is obtained from the whole population.\n</p><p>Other examples of statistics include\n</p>\n<ul><li><a class="mw-redirect" class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Statistic&amp;action=edit&amp;section=2" title="Edit section: Properties">edit</a><span class="mw-editsection-bracket">]</span></span></h2>\n<h3><span class="mw-headline" id="Observability">Observability</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Statistic&amp;action=edit&amp;section=3" title="Edit section: Observability">edit</a><span class="mw-editsection-bracket">]</span></span></h3>\n<p>A statistic is an <i>observable</i> <a href="/wiki/Random_variable" title="Random variable">random variable</a>, which differentiates it both from a <i><a href="/wiki/Statistical_parameter" title="Statistical parameter">parameter</a></i> that is a generally unobservable quantity describing a property of a <a href="/wiki/Statistical_population" title="Statistical population">statistical population</a>, and from an unobservable random variable, such as the difference between an observed measurement and a population average. A parameter can only be computed exactly if entire population can be observed without error; for instance, in a perfect census or for a population of <a href="/wiki/Standardized_test" title="Standardized test">standardized test</a> takers.\n</p><p>Statisticians often contemplate a <a class="mw-redirect" href="/wiki/Parameterized_family" title="Parameterized family">parameterized family</a> of <a href="/wiki/Probability_distribution" title="Probability distribution">probability distributions</a>, any member of which could be the distribution of some measurable aspect of each member of a population, from which a sample is drawn randomly. For example, the parameter may be the average height of 25-year-old men in North America. The height of the members of a sample of 100 such men are measured; the average of those 100 numbers is a statistic. The average of the heights of all members of the population is not a statistic unless that has somehow also been ascertained (such as by measuring every member of the population).  The average height that would be calculated using <i>all</i> of the individual heights of <i>all</i> 25-year-old North American men is a parameter, and not a statistic.\n</p>\n<h3><span class="mw-headline" id="Statistical_properties">Statistical properties</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Statistic&amp;action=edit&amp;section=4" title="Edit section: Statistical properties">edit</a><span class="mw-editsection-bracket">]</span></span></h3>\n<p>Important potential properties of statistics include <a href="/wiki/Completeness_(statistics)" title="Completeness (statistics)">completeness</a>, <a href="/wiki/Consistent_estimator" title="Consistent estimator">consistency</a>, <a class="mw-redirect" href="/wiki/Sufficiency_(statistics)" title="Sufficiency (statistics)">sufficiency</a>, <a class="mw-redirect" href="/wiki/Estimator_bias" title="Estimator bias">unbiasedness</a>, <a href="/wiki/Minimum_mean_square_error" title="Minimum mean square error">minimum mean square error</a>, low <a href="/wiki/Variance" title="Variance">variance</a>, <a href="/wiki/Robust_statistics" title="Robust statistics">robustness</a>, and computational convenience.\n</p>\n<h3><span class="mw-headline" id="Information_of_a_statistic">Information of a statistic</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Statistic&amp;action=edit&amp;section=5" title="Edit section: Information of a statistic">edit</a><span class="mw-editsection-bracket">]</span></span></h3>\n<p>Information of a statistic on model parameters can be defined in several ways. The most common is the <a href="/wiki/Fisher_information" title="Fisher information">Fisher information</a>, which is defined on the statistic model induced by the statistic.  <a class="mw-redirect"
