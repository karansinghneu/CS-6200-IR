if <i>q</i> is quoted in integer units of the elementary charge and the terminal bias in volts, one gets an energy in eV.\n</p><p>Like the elementary charge on which it is based, it is not an independent quantity but is equal to <span class="nowrap">1 <span class="sfrac nowrap" style="display:inline-block; vertical-align:-0.5em; font-size:85%; text-align:center;"><span style="display:block; line-height:1em; margin:0 0.1em;">J</span><span class="visualhide">/</span><span style="display:block; line-height:1em; margin:0 0.1em; border-top:1px solid;">C</span></span>&#8201;<span class="nowrap">&#8730;<span style="border-top:1px solid; padding:0 0.1em;">2<a href="/wiki/Planck_constant" title="Planck constant"><i>h</i></a><a class="mw-redirect" href="/wiki/Fine_structure_constant" title="Fine structure constant"><i>&#945;</i></a>&#8201;/&#8201;<a class="mw-redirect" href="/wiki/Magnetic_constant" title="Magnetic constant"><i>&#956;</i><sub>0</sub></a><a href="/wiki/Speed_of_light" title="Speed of light"><i>c</i><sub>0</sub></a></span></span></span>. It is a common <a class="mw-redirect" href="/wiki/Unit_of_energy" title="Unit of energy">unit of energy</a> within physics, widely used in <a href="/wiki/Solid-state_physics" title="Solid-state physics">solid state</a>, <a href="/wiki/Atomic_physics" title="Atomic physics">atomic</a>, <a href="/wiki/Nuclear_physics" title="Nuclear physics">nuclear</a>, and <a href="/wiki/Particle_physics" title="Particle physics">particle physics</a>. It is commonly used with the <a href="/wiki/Metric_prefix" title="Metric prefix">metric prefixes</a> milli-, kilo-, mega-, giga-, tera-, peta- or exa- (meV, keV, MeV, GeV, TeV, PeV and EeV respectively). In some older documents, and in the name <a href="/wiki/Bevatron" title="Bevatron">Bevatron</a>, the symbol BeV is used, which stands for <a href="/wiki/Billion" title="Billion">billion</a>
