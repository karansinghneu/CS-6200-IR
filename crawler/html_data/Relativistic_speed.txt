Relativistic effects are those discrepancies between values calculated by models considering and not considering <a href="/wiki/Theory_of_relativity" title="Theory of relativity">relativity</a>.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> Related words are <a href="/wiki/Velocity" title="Velocity">velocity</a>, <a href="/wiki/Rapidity" title="Rapidity">rapidity</a>, and <a href="/wiki/Proper_velocity" title="Proper velocity">celerity</a> which is <a href="/wiki/Proper_velocity" title="Proper velocity">proper velocity</a>. <a href="/wiki/Speed" title="Speed">Speed</a> is a <a href="/wiki/Scalar_(mathematics)" title="Scalar (mathematics)">scalar</a>, being the <a class="mw-redirect" href="/wiki/Magnitude_(vector)" title="Magnitude (vector)">magnitude</a> of the velocity <a href="/wiki/Vector_(mathematics_and_physics)" title="Vector (mathematics and physics)">vector</a> which in relativity is the <a href="/wiki/Four-velocity" title="Four-velocity">four-velocity</a> and in three-dimension Euclidean space a three-velocity. Speed is <a href="/wiki/Empirical_research" title="Empirical research">empirically</a> measured as average speed, although <a href="/wiki/Lidar" title="Lidar">current devices in common use</a> can estimate speed over very small <a href="/wiki/Interval_(mathematics)" title="Interval (mathematics)">intervals</a> and closely approximate <a href="/wiki/Instant" title="Instant">instantaneous</a> speed. Non-relativistic discrepancies include <a class="mw-redirect" href="/wiki/Cosine_error_effect" title="Cosine error effect">cosine error</a> which occurs in <a href="/wiki/Radar_gun" title="Radar gun">speed detection devices</a> when only one scalar component of the three-velocity is measured and the <a href="/wiki/Doppler_effect" title="Doppler effect">Doppler effect</a> which may affect observations of wavelength and frequency.</p><div class="center"><div class="thumb tnone"><div class="thumbinner" style="width:356px;"><a class="image" href="/wiki/File:Lorentz_factor.svg"><img alt="" class="thumbimage" data-file-height="1118" data-file-width="1102" decoding="async" height="359" src="//upload.wikimedia.org/wikipedia/commons/thumb/0/00/Lorentz_factor.svg/354px-Lorentz_factor.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/0/00/Lorentz_factor.svg/531px-Lorentz_factor.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/0/00/Lorentz_factor.svg/708px-Lorentz_factor.svg.png 2x" width="354"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/File:Lorentz_factor.svg" title="Enlarge"></a></div>Lorentz factor, <i>&#947;,</i> as a function of speed, <i>v</i>. Its initial value is 1 when speed is zero and increases without bound as speed approaches light speed, <i>c</i>.</div></div></div></div><div class="center"><div class="thumb tnone"><div class="thumbinner" style="width:348px;"><a class="image" href="/wiki/File:Lorentz_factor_inverse.svg"><img alt="" class="thumbimage" data-file-height="1000" data-file-width="1000" decoding="async" height="346" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/35/Lorentz_factor_inverse.svg/346px-Lorentz_factor_inverse.svg.png" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/35/Lorentz_factor_inverse.svg/519px-Lorentz_factor_inverse.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/3/35/Lorentz_factor_inverse.svg/692px-Lorentz_factor_inverse.svg.png 2x" width="346"/></a> <div class="thumbcaption"><div class="magnify"><a class="internal" href="/wiki/File:Lorentz_factor_inverse.svg" title="Enlarge"></a></div>Inverse of Lorentz factor as a function of speed, <i>v</i>, as a proportion of light speed, <i>c</i> - a circular arc.</div></div></div></div><p>Relativistic effects are highly <a href="/wiki/Nonlinear_system" title="Nonlinear system">non-linear</a> and for everyday purposes are insignificant because the <a href="/wiki/Newton%27s_laws_of_motion" title="Newton\'s laws of motion">Newtonian</a> model closely approximates the relativity model. In special relativity the <a href="/wiki/Lorentz_factor" title="Lorentz factor">Lorentz factor</a> is a measure of <a href="/wiki/Time_dilation" title="Time dilation">time dilation</a>, <a href="/wiki/Length_contraction" title="Length contraction">length contraction</a> and relativistic <a class="mw-redirect" href="/wiki/Relativistic_Mass" title="Relativistic Mass">mass</a> increase of a moving object.\n</p><h2><span class="mw-headline" id="See_also">See also</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a
