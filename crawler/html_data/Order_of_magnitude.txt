It is equal to the <a href="/wiki/Logarithm" title="Logarithm">logarithm</a> (base 10) rounded to a <a href="/wiki/Integer" title="Integer">whole number</a>. order of magnitude</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Order_of_magnitude&amp;action=edit&amp;section=3" title="Edit section: Calculating the order of magnitude">edit</a><span class="mw-editsection-bracket">]</span></span></h3>\n<p>The order of magnitude of a number is, intuitively speaking, the number of powers of 10 contained in the number.  More precisely, the order of magnitude of a number can be defined in terms of the <a href="/wiki/Common_logarithm" title="Common logarithm">common logarithm</a>, usually as the <a href="/wiki/Integer" title="Integer">integer</a> part of the logarithm, obtained by <a href="/wiki/Truncation" title="Truncation">truncation</a>.  For example, the number <span class="nowrap"><span class="sortkey" style="display:none">7006400000000000000&#9824;</span>4<span style="margin-left:.25em;">000</span><span style="margin-left:.25em;">000</span></span> has a logarithm (in base 10) of 6.602; its order of magnitude is 6.  When truncating, a number of this order of magnitude is between 10<sup>6</sup> and 10<sup>7</sup>. In a similar example, with the phrase "He had a seven-figure income", the order of magnitude is the number of figures minus one, so it is very easily determined without a calculator to 6. An order of magnitude is an approximate position on a <a href="/wiki/Logarithmic_scale" title="Logarithmic scale">logarithmic scale</a>.\n</p>\n<h3><span class="mw-headline" id="Order-of-magnitude_estimate">Order-of-magnitude estimate</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Order_of_magnitude&amp;action=edit&amp;section=4" title="Edit section: Order-of-magnitude estimate">edit</a><span class="mw-editsection-bracket">]</span></span></h3>\n<p>An order-of-magnitude estimate of a variable, whose precise value is unknown, is an estimate <a href="/wiki/Rounding" title="Rounding">rounded</a> to the nearest power of ten.  For example, an order-of-magnitude estimate for a variable between about 3 billion and 30 billion (such as the <a href="/wiki/Human" title="Human">human</a> <a href="/wiki/Population" title="Population">population</a> of the <a href="/wiki/Earth" title="Earth">Earth</a>) is 10 <a class="mw-redirect" href="/wiki/1000000000_(number)" title="1000000000 (number)">billion</a>. To round a number to its nearest order of magnitude, one rounds its logarithm to the nearest integer.  Thus <span class="nowrap"><span class="sortkey" style="display:none">7006400000000000000&#9824;</span>4<span style="margin-left:.25em;">000</span><span style="margin-left:.25em;">000</span></span>, which has a logarithm (in base 10) of 6.602, has 7 as its nearest order of magnitude, because "nearest" implies rounding rather than truncation.  For a number written in scientific notation, this logarithmic rounding scale requires rounding up to the next power of ten when the multiplier is greater than the square root of ten (about 3.162). For example, the nearest order of magnitude for <span class="nowrap"><span class="sortkey" style="display:none">7008170000000000000&#9824;</span>1.7<span style="margin-left:0.25em;margin-right:0.15em;">&#215;</span>10<sup>8</sup></span> is 8, whereas the nearest order of magnitude for <span class="nowrap"><span class="sortkey" style="display:none">7008370000000000000&#9824;</span>3.7<span style="margin-left:0.25em;margin-right:0.15em;">&#215;</span>10<sup>8</sup></span> is 9. An order-of-magnitude estimate is sometimes also called a <a class="mw-redirect" href="/wiki/Zeroth_order_approximation" title="Zeroth order approximation">zeroth order approximation</a>.\n</p>\n<h3><span class="mw-headline" id="Order_of_magnitude_difference">Order of magnitude difference</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Order_of_magnitude&amp;action=edit&amp;section=5" title="Edit section: Order of magnitude difference">edit</a><span class="mw-editsection-bracket">]</span></span></h3>\n<p>An order-of-magnitude difference between two values is a factor of 10.  For example, the mass of the planet <a href="/wiki/Saturn" title="Saturn">Saturn</a> is 95 times that of <a href="/wiki/Earth" title="Earth">Earth</a>, so Saturn is <i>two orders of magnitude</i> more massive than Earth.  Order-of-magnitude differences are called <b><a href="/wiki/Decade_(log_scale)" title="Decade (log scale)">decades</a></b> when measured on a <a href="/wiki/Logarithmic_scale" title="Logarithmic scale">logarithmic scale</a>.\n</p>\n<h2><span class="mw-headline" id="Non-decimal_orders_of_magnitude">Non-decimal orders of magnitude</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Order_of_magnitude&amp;action=edit&amp;section=6" title="Edit section: Non-decimal orders of magnitude">edit</a><span class="mw-editsection-bracket">]</span></span></h2>\n<div class="hatnote navigation-not-searchable" role="note">See also: <a href="/wiki/Logarithmic_scale" title="Logarithmic scale">Logarithmic scale</a></div>\n<p>Other orders of magnitude may be calculated using <a href="/wiki/Radix" title="Radix">bases</a> other than 10.
