name given to two distinct series of <a href="/wiki/Soviet_Union" title="Soviet Union">Soviet</a> <a href="/wiki/Robotic_spacecraft" title="Robotic spacecraft">unmanned space program</a> undertaken from 1964 to 1970. The first series, based on the <a class="mw-redirect" href="/wiki/3MV_planetary_probe" title="3MV planetary probe">3MV planetary probe</a>, was intended to gather information about nearby <a class="mw-redirect" href="/wiki/Planets" title="Planets">planets</a>. The second series of test spacecraft, being a precursor to manned circumlunar loop flights, used a stripped-down variant of <a class="mw-redirect" href="/wiki/Soyuz_spacecraft" title="Soyuz spacecraft">Soyuz spacecraft</a>, consisting of the <a class="mw-redirect" href="/wiki/Soyuz_spacecraft#Service_Module" title="Soyuz spacecraft">service</a> and <a class="mw-redirect" href="/wiki/Soyuz_spacecraft#Reentry_Module" title="Soyuz spacecraft">descent modules</a>, but lacking the <a class="mw-redirect" href="/wiki/Soyuz_spacecraft#Orbital_Module" title="Soyuz spacecraft">orbital module</a>. Two tortoises and other lifeforms aboard <a href="/wiki/Zond_5" title="Zond 5">Zond 5</a> were the first to travel around the Moon and return to Earth.\n</p>\n<div class="toc" id="toc"><input title="3MV planetary probe">3MV planetary probe</a>, intended to explore <a href="/wiki/Venus" title="Venus">Venus</a> and <a href="/wiki/Mars" title="Mars">Mars</a>. After two failures, <a href="/wiki/Zond_3" title="Zond 3">Zond 3</a> was sent on a test mission, becoming the second spacecraft to photograph the far side of the Moon.  It then continued out to the orbit of Mars in order to test telemetry and spacecraft systems.\n</p>\n<h2><span class="mw-headline" id="Circumlunar_missions">Circumlunar missions</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Zond_program&amp;action=edit&amp;section=2" title="Edit section: Circumlunar missions">edit</a><span class="mw-editsection-bracket">]</span></span></h2>\n<div class="hatnote navigation-not-searchable" role="note">Main articles: <a href="/wiki/Soyuz_7K-L1" title="Soyuz 7K-L1">Soyuz 7K-L1</a> and <a class="mw-redirect" href="/wiki/Moon_Race#Onward_to_the_Moon" title="Moon Race">Moon Race &#167;&#160;Onward to the Moon</a></div>\n<p>The missions <a href="/wiki/Zond_4" title="Zond 4">4</a> through <a href="/wiki/Zond_8" title="Zond 8">8</a> were test flights under for the <a class="mw-redirect" href="/wiki/Soviet_Moonshot" title="Soviet Moonshot">Soviet Moonshot</a> during the <a class="mw-redirect" href="/wiki/Moon_race" title="Moon race">Moon race</a>. The <a href="/wiki/Soyuz_7K-L1" title="Soyuz 7K-L1">Soyuz 7K-L1</a> (also mentioned just as L1) spacecraft was used for the Moon-aimed missions, stripped down to make it possible to launch around the Moon from the Earth. They were launched on the <a class="mw-redirect" href="/wiki/Proton_rocket" title="Proton rocket">Proton rocket</a> which was just powerful enough to send the Zond on a <a class="mw-redirect" href="/wiki/Free_return_trajectory" title="Free return trajectory">free return trajectory</a> around the Moon without going into lunar orbit (the type path that <a href="/wiki/Apollo_13" title="Apollo 13">Apollo 13</a> flew in its emergency abort). With minor modification, Zond was capable of carrying two <a class="mw-redirect" href="/wiki/Cosmonauts" title="Cosmonauts">cosmonauts</a>.\n</p><p>In the beginning there were serious reliability problems with both the new Proton rocket and the similarly new Soyuz spacecraft, but the test flights pressed ahead with some glitches. The majority of test flights from 1967&#8211;1970 (<a href="/wiki/Zond_4" title="Zond 4">Zond 4</a> to <a href="/wiki/Zond_8" title="Zond 8">Zond 8</a>) showed problems during re-entry.\n</p><p>The Zond spacecraft made only unmanned automatic flights. Four of these suffered malfunctions that would have injured or killed any crew. Instrumentation flown on these missions gathered data on <a class="mw-redirect" href="/wiki/Micrometeor" title="Micrometeor">micrometeor</a> flux, solar and <a href="/wiki/Cosmic_ray" title="Cosmic ray">cosmic rays</a>, <a href="/wiki/Magnetic_field" title="Magnetic field">magnetic fields</a>, radio emissions, and <a href="/wiki/Solar_wind" title="Solar wind">solar wind</a>. Many photographs were taken and biological payloads were also flown.\n</p>\n<h2><span class="mw-headline" id="Timetable">Timetable</span><span class="mw-editsection"><span
