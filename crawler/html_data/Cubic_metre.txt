href="/wiki/Cube_(geometry)" title="Cube (geometry)">cube</a> with edges one <a href="/wiki/Metre" title="Metre">metre</a> in length.  An alternative name, which allowed a different usage with <a href="/wiki/Metric_prefix" title="Metric prefix">metric prefixes</a>, was the <a class="mw-redirect" href="/wiki/St%C3%A8re" title="St&#232;re">st&#232;re</a>, still sometimes used for <a href="/wiki/Dry_measure" title="Dry measure">dry measure</a> (for instance, in reference to <a href="/wiki/Wood" title="Wood">wood</a>).
