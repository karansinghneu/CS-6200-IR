title="Physics">physics</a>, <a href="/wiki/Astronomy" title="Astronomy">astronomy</a>, <a href="/wiki/Chemistry" title="Chemistry">chemistry</a>, <a href="/wiki/Biology" title="Biology">biology</a> and <a href="/wiki/Geography" title="Geography">geography</a>, <b>number density</b> (symbol: <i>n</i> or <i>&#961;</i><sub>N</sub>) is an <a class="mw-redirect" href="/wiki/Intensive_quantity" title="Intensive quantity">intensive quantity</a> used to describe the degree of <a href="/wiki/Concentration" title="Concentration">concentration</a> of <a class="mw-redirect" href="/wiki/Countable" title="Countable">countable</a> objects (<a href="/wiki/Elementary_particle" title="Elementary particle">particles</a>, <a class="mw-redirect" href="/wiki/Molecules" title="Molecules">molecules</a>, href="/wiki/Three-dimensional_space" title="Three-dimensional space">three-dimensional</a> volumetric number density, <a href="/wiki/Two-dimensional_space" title="Two-dimensional space">two-dimensional</a> areal number density, or <a href="/wiki/One-dimensional_space" title="One-dimensional space">one-dimensional</a> linear number density. <a href="/wiki/Population_density" title="Population density">Population density</a> is an example of areal number density. The term <b>number concentration</b> (symbol: <i>C</i>, to avoid confusion with <a href="/wiki/Amount_of_substance" title="Amount of substance">amount of substance</a> <i>n</i>) is sometimes used in chemistry for the same quantity, particularly when comparing with other concentrations.\n</p>\n<div class="toc" id="toc"><input class="toctogglecheckbox" id="toctogglecheckbox" role="button" class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Number_density&amp;action=edit&amp;section=1" title="Edit section: Definition">edit</a><span class="mw-editsection-bracket">]</span></span></h2>\n<p>Volume number density is the number of specified objects per unit <a href="/wiki/Volume" title="Volume">volume</a>:<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>\n</p>\n<dl><dd><span class="mwe-math-element"><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;"><math alttext="{\\displaystyle n={\\frac {N}{V}},}" xmlns="http://www.w3.org/1998/Math/MathML">\n<semantics>\n<mrow class="MJX-TeXAtom-ORD">\n<mstyle displaystyle="true" scriptlevel="0">\n<mi>n</mi>\n<mo>=</mo>\n<mrow class="MJX-TeXAtom-ORD">\n<mfrac>\n<mi>N</mi>\n<mi>V</mi>\n</mfrac>\n</mrow>\n<mo>,</mo>\n</mstyle>\n</mrow>\n<annotation encoding="application/x-tex">{\\displaystyle n={\\frac {N}{V}},}</annotation>\n</semantics>\n</math></span><img alt="n={\\frac  {N}{V}}," aria-hidden="true" class="mwe-math-fallback-image-inline" <i>N</i> is the total number of objects in a volume <i>V</i>. \n</p><p>Here it is assumed<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> that <i>N</i> is large enough that <a href="/wiki/Rounding" title="Rounding">rounding</a> of the count to the nearest <a href="/wiki/Integer" title="Integer">integer</a> does not introduce much of an <a class="mw-redirect" href="/wiki/Random_error" title="Random error">error</a>, however <i>V</i> is chosen to be small enough that the resulting <i>n</i> does not depend much on the <a href="/wiki/Volume" title="Volume">size</a> or <a href="/wiki/Shape" title="Shape">shape</a> of the volume <i>V</i>.\n</p>\n<h2><span class="mw-headline" id="Units">Units</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Number_density&amp;action=edit&amp;section=2" title="Edit section: Units">edit</a><span class="mw-editsection-bracket">]</span></span></h2>\n<p>In <a href="/wiki/International_System_of_Units" title="International System of Units">SI</a> units, number density is measured in m<sup>&#8722;3</sup>, although cm<sup>&#8722;3</sup> is often used. However, these units are not quite practical when dealing with atoms or molecules of <a href="/wiki/Gas" title="Gas">gases</a>, <a href="/wiki/Liquid" title="Liquid">liquids</a> or <a href="/wiki/Solid" title="Solid">solids</a> at <a href="/wiki/Room_temperature" title="Room temperature">room temperature</a> and <a href="/wiki/Atmospheric_pressure" title="Atmospheric pressure">atmospheric pressure</a>, because the resulting numbers are extremely large (on the order of 10<sup>20</sup>). Using the number density of an <a href="/wiki/Ideal_gas" title="Ideal gas">ideal gas</a> at <span class="nowrap">0 <a href="/wiki/Celsius" title="Celsius">&#176;C</a></span> and <span class="nowrap">1 <a href="/wiki/Atmosphere_(unit)" title="Atmosphere (unit)">atm</a></span> as a <a class="mw-redirect" href="/wiki/Yardstick" title="Yardstick">yardstick</a>: <span class="nowrap"><a href="/wiki/Loschmidt_constant" title="Loschmidt constant">n<sub>0</sub></a> = 1 <a href="/wiki/Amagat" title="Amagat">amg</a> = 2.686,777,4 &#215; 10<sup>25</sup> m<sup>&#8722;3</sup></span> is often introduced as a unit of number density, for any substances at any conditions (not necessarily limited to an ideal gas at <span class="nowrap">0 &#176;C</span> and <span class="nowrap">1 atm</span>).<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup>\n</p>\n<h2><span class="mw-headline" id="Usage">Usage</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Number_density&amp;action=edit&amp;section=3" title="Edit section: Usage">edit</a><span class="mw-editsection-bracket">]</span></span></h2>\n<p>Using the number density as a <a href="/wiki/Function_(mathematics)" title="Function (mathematics)">function</a> of <a href="/wiki/Coordinate_system" title="Coordinate system">spatial coordinates</a>, the total number of objects <i>N</i> in the entire volume <i>V</i> can be calculated as\n</p>\n<dl><dd><span class="mwe-math-element"><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;"><math alttext="{\\displaystyle N=\\iiint _{V}n(x,\\,y,\\,z)\\,\\mathrm {d} V,}" xmlns="http://www.w3.org/1998/Math/MathML">\n<semantics>\n<mrow class="MJX-TeXAtom-ORD">\n<mstyle displaystyle="true" scriptlevel="0">\n<mi>N</mi>\n<mo>=</mo>\n<msub>\n<mo>&#8749;<!-- &#8749;
