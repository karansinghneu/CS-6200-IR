id="siteSub">From Wikipedia, the free encyclopedia</div> <div id="contentSub"></div>\n<div id="jump-to-nav"></div> <a class="mw-jump-link" href="#mw-head">Jump to navigation</a>\n<a class="mw-jump-link" href="#p-search">Jump to search</a>\n<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><div class="mw-parser-output"><p>In <a href="/wiki/Science" title="Science">science</a>, a <b>null result</b> is a result without the expected content: that is, the proposed result is absent.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup> It is an experimental outcome which does not show an otherwise expected effect. This does not imply a result of zero or nothing, simply a result that does not support the <a href="/wiki/Hypothesis" title="Hypothesis">hypothesis</a>. The term is a translation of the scientific <a href="/wiki/Latin" title="Latin">Latin</a> <i>nullus resultarum</i>, meaning "no consequence".\n</p><p>In <a href="/wiki/Statistical_hypothesis_testing" title="Statistical hypothesis testing">statistical hypothesis testing</a>, a null result occurs when an experimental result is not significantly different from what is to be expected under the <a href="/wiki/Null_hypothesis" title="Null hypothesis">null hypothesis</a>; its probability (under the null hypothesis) does not exceed the <a class="mw-redirect" href="/wiki/Significance_level" title="Significance level">significance level</a>, i.e., the threshold set prior to testing for rejection of the null hypothesis. The significance level varies, but is often set at <a href="/wiki/P-value" title="P-value"><i>p</i>-value</a> 0.05 (5%).\n</p><p>As an example in <a href="/wiki/Physics" title="Physics">physics</a>, the results of the <a href="/wiki/Michelson%E2%80%93Morley_experiment" title="Michelson&#8211;Morley experiment">Michelson&#8211;Morley experiment</a> were of this type, as it did not detect the expected velocity relative to the postulated <a href="/wiki/Luminiferous_aether" title="Luminiferous aether">luminiferous aether</a>. This experiment\'s famous failed detection, commonly referred to as the <i>null result</i>, contributed to the development of <a href="/wiki/Special_relativity" title="Special relativity">special relativity</a>. The experiment did appear to measure a non-zero "drift", but the value was far too small to account for the theoretically expected results; it is generally thought to be inside the <a class="mw-redirect" href="/wiki/Noise_level" title="Noise level">noise level</a> of the experiment.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup>\n</p>\n<h2><span class="mw-headline" id="Scientific_journals_for_null_results">Scientific journals for null results</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Null_result&amp;action=edit&amp;section=1" title="Edit section: Scientific journals for null results">edit</a><span class="mw-editsection-bracket">]</span></span></h2>\n<p>There are now several scientific journals dedicated to the publication of negative or null results, including the following:\n</p>\n<ul><li><a class="new" href="/w/index.php?title=Data_-_the_journal_for_Negative,_Null_and_Inconclusive_results_(NNI_results)&amp;action=edit&amp;redlink=1" title="Data -
